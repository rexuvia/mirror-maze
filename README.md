# 🔮 Mirror Maze - Harmonic Resonance

A beautiful laser beam puzzle game featuring the unique **Harmonic Resonance** mechanic, where intersecting beams create powerful resonance nodes that emit special harmonic beams.

## 🎮 Gameplay

**Core Mechanic:**
- Place and rotate mirrors to bounce laser beams
- Hit all targets in each level to progress
- Click/tap mirrors to rotate them 45° at a time

**Harmonic Resonance System:**
- When 2+ laser beams cross paths, they create a **resonance node**
- Resonance nodes emit 8-directional harmonic beams (purple-gold gradient)
- Harmonic beams have special properties:
  - Pass through barriers that block normal beams
  - Can hit harmonic targets (purple star markers)
  - Create beautiful particle effects and glowing intersections
- More beams intersecting = stronger resonance (brighter glow, enhanced effects)

## 🎯 Progression

The game features 5 carefully designed levels:

1. **Tutorial** - Learn basic beam reflection with mirrors
2. **First Resonance** - Create your first beam intersection to hit harmonic targets
3. **Multi-Resonance** - Orchestrate multiple intersections to hit both normal and harmonic targets
4. **Barrier Puzzle** - Use harmonic beams to pass through barriers
5. **Complex Harmony** - Master complex beam paths with multiple barriers and targets

Progress is automatically saved in your browser's localStorage.

- Plan your mirror placement to create intersections where needed

- Experiment with different angles - mirrors rotate in 45° increments
- More powerful resonances (3+ beams) create stronger visual feedback
